## 考試資訊

| 項目 | 內容 |
| --- | --- |
| 題數 | 65 題（50 題計分 + 15 題不計分測試題） |
| 時間 | 90 分鐘 |
| 及格分數 | 700 / 1000（scaled score，範圍 100–1000） |
| 題型 | 單選 + 複選；closed book，不可看任何參考資料 |
| 費用 | USD $100 |
| 有效期 | 3 年（重考 CLF-C02 或通過更高階認證可續期） |

> 💡 15 題不計分題不會標示出來，每題都當真題作答即可；複選題一定會明確告訴你要選幾個。

## 四大考試領域

| 領域 | 名稱 | 占比 |
| --- | --- | --- |
| Domain 1 | 雲端概念（Cloud Concepts） | 24% |
| Domain 2 | 安全與合規（Security & Compliance） | 30% |
| Domain 3 | 雲端技術與服務（Cloud Technology & Services） | 34% |
| Domain 4 | 計費、定價與支援（Billing, Pricing & Support） | 12% |

> 🚨 **複習重心**：Domain 3（34%）最大，Domain 2（安全）由舊版 CLF-C01 的 25% 提高到 30%，是改版後最大的變化。兩者合計 **64%**，時間要優先投資在這裡。

## 模組目錄

| 模組 | 標題 | 主要領域 |
| --- | --- | --- |
| **M1** | 雲端入門：雲端運算、全球架構與責任共擔 | 雲端概念 + 安全 |
| **M2** | 雲端運算 EC2：執行個體、計費、擴展與訊息 | 技術與服務 |
| **M3** | 探索運算服務：Lambda、容器、Step Functions | 技術與服務 |
| **M4** | 走向全球：Region 選擇、多 AZ 可靠性與 IaC | 技術與服務 + 雲端概念 |
| **M5** | 雲端網路：VPC、子網、安全群組與全球網路 | 技術與服務 + 安全 |
| **M6** | 儲存：S3、EBS、EFS 與其他儲存服務 | 技術與服務 |
| **M7** | 資料庫：RDS、DynamoDB 與其他資料庫 | 技術與服務 |
| **M8** | AI、ML 與資料分析 | 技術與服務 |
| **M9** | 雲端安全：IAM、加密、威脅偵測與合規 | 安全與合規 |
| **M10** | 監控、合規與治理 | 安全與合規 + 技術 |
| **M11** | 定價、計費與支援方案 | 計費與支援 |
| **M12** | 遷移上雲：策略、工具與 CAF | 雲端概念 |
| **M13** | 架構完善方案：Well-Architected Framework | 雲端概念 |

## 建議學習路徑

- **完全新手**：依 M1 → M13 順序閱讀，先建立雲端概念與責任共擔模型，再進入各服務細節。
- **有 IT 基礎**：概念模組（M1、M4、M12、M13）速讀，重點放在技術服務模組（M5 網路、M6 儲存、M7 資料庫、M3 運算）。
- **考前衝刺**：依領域占比優先複習 Domain 3 → Domain 2，利用每章結尾的「考試重點整理」做最後快掃。
